Client: React, React Router

Server: Node.js, Express.js, Mongoose (MongoDB)

### Project setup

#### Client
```
cd client
```
```
npm install
```

#### Server
```
cd server
```
```
npm install
```

### Compiles for development

#### Client
```
cd client
```
```
npm run start
```

#### Server
```
cd server
```
```
npm run dev
```

### Compiles for production

#### Client
```
cd client
```
```
npm run build
```

#### Server
```
cd server
```
```
npm run start
```
